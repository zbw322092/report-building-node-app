## Polyfill Libraries
Imagining that, if we developing a PC frontend application, which just requires some simple "smooth skip to" effect, it would be crazy to include the whole `iScroll` core build file.
scroll-behavior: smooth helps with compatibility problems.
Now, polyfill comes in.
smoothscroll
The library implements window.scroll, window.scrollTo, window.scrollBy, Element.prototype.scroll, Element.prototype.scrollTo, Element.prototype.scrollBy, Element.prototype.scrollIntoView polyfills, with smooth scrolling effects.
